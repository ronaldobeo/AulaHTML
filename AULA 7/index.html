Aula 07 - Estudo de CSS

A sigla CSS é Cascading Style Sheets é uma liguagem para estilizar e/ou diagramar elementos HTML dentro de uma pagina web

O CSS funciona em conjunto com o HTML, enquanto o HTML diz o que será exibido nas paginas, o CSS diz como deverá ser exibido.
